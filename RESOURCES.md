# AI Engineering Resources

## Knowledge

- [OpenAI API docs: Agents SDK](https://developers.openai.com/api/docs/guides/agents)
  Use for: tool-using agents, orchestration, state, guardrails, tracing, and when to use SDK-owned orchestration.
- [OpenAI API docs: Working with evals](https://developers.openai.com/api/docs/guides/evals)
  Use for: learning the evaluation loop and grader vocabulary. Note: the legacy Evals platform has a published deprecation timeline, so keep the pattern portable.
- [OpenAI API docs: Production best practices](https://developers.openai.com/api/docs/guides/production-best-practices)
  Use for: staging projects, secrets, scaling, rate limits, caching, cost tracking, and production readiness.
- [OpenAI API docs: Safety best practices](https://developers.openai.com/api/docs/guides/safety-best-practices)
  Use for: moderation, adversarial testing, human review, and misuse controls.
- [OpenAI API docs: Latency optimization](https://developers.openai.com/api/docs/guides/latency-optimization)
  Use for: model choice, reducing generated tokens, reducing input tokens, parallelism, and non-LLM fallbacks.
- [OpenAI API docs: Cost optimization](https://developers.openai.com/api/docs/guides/cost-optimization)
  Use for: fewer requests, fewer tokens, smaller models, batch work, and async lower-priority processing.
- [Hugging Face LLM Course](https://huggingface.co/learn/llm-course/chapter1/1)
  Use for: transformer and LLM foundations, tokenizers, datasets, and model ecosystem literacy.
- [Hugging Face AI Agents Course](https://huggingface.co/learn/agents-course/unit0/introduction)
  Use for: agent concepts and comparison of agent frameworks.
- [LlamaIndex documentation](https://developers.llamaindex.ai/python/framework/)
  Use for: RAG, ingestion, indexing, query engines, chat engines, agents, and evaluation integrations.
- [LangGraph documentation](https://docs.langchain.com/oss/python/langgraph/overview)
  Use for: long-running stateful workflows, persistence, human-in-the-loop, streaming, and agent orchestration.

## Wisdom (Communities)

- [OpenAI Developer Community](https://community.openai.com/)
  Use for: platform-specific API behavior, migration issues, and production edge cases.
- [Hugging Face Forums](https://discuss.huggingface.co/)
  Use for: open-source model, dataset, and deployment questions.
- [LangChain Forum](https://forum.langchain.com/)
  Use for: LangGraph and LangChain implementation questions.
- [LlamaIndex GitHub Discussions](https://github.com/run-llama/llama_index/discussions)
  Use for: RAG implementation questions and framework-specific troubleshooting.
